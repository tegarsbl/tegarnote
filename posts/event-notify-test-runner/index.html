Event Notify Test Runner

Published: 05 Jan 2025

ENTR (Event Notify Test Runner) adalah alat yang simpel tapi sangat berguna buat ngejalanin perintah otomatis setiap kali ada perubahan di file atau direktori tertentu.

Cara Kerja ENTR

Secara umum, entr bakal “ngawasin” file atau direktori, dan begitu ada perubahan (kayak di-save ulang, di-edit, atau ditambahin), entr otomatis ngejalanin perintah yang kita tentukan. Ini ngebantu banget buat workflow yang butuh banyak revisi, seperti coding, nulis dokumen, atau kerja di proyek yang perlu cek hasil secara berkala.

Contoh Penggunaan:

$ echo main.py | entr python main.py

Kelebihan ENTR

  • Hemat Waktu: Kamu nggak perlu compile atau jalankan script secara manual terus-menerus.
  • Simple dan Ringan: entr itu sangat ringan dan bisa langsung dipake tanpa konfigurasi ribet.
  • Banyak Fungsi: Bisa dipakai buat compile kode, running unit test, nulis dokumentasi, dll.

Jadi, entr ini cocok banget buat kalian yang sering coding atau nulis dokumen.
